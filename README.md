
# DALL-E 2.0 - Based on OpenAI's GPT3




## Run Locally

Clone the project

```bash
  git clone https://github.com/Ashish0926/dalle2.0.git
```

Go to the project directory

```bash
  cd dalle2.0
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

```bash
  https://dalle-2.netlify.app/
```
